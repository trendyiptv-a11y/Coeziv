Model Coeziv 3.14 & Dinamica 2π — Reformulare operațională pentru cercetare

Scop: transformarea teoriei originale într-un cadru operațional, cuantificabil și falsificabil — gata pentru implementare experimentală și simulare.

0. Principii generale

  • Modelul este fenomenologic. Trebuie declarat explicit ca atare: nu este o derivare microscopică dintr-un Hamiltonian, ci un indice sintetic care reunește observabile macroscopice și spectroscopice.
  • Toate mărimile au unități și proceduri clare de măsurare.
  • Predicțiile principale sunt falsificabile numeric (ex: test statistic pentru R ≈ π cu interval de toleranță δ).

1. Notare și definiții canonice

  • T — temperatură (K). Când este relevant, vom arăta și valorile în °C.
  • ρ(T) — densitate macroscopică (kg·m⁻3), măsurată conform standardelor (IAPWS).
  • r(T) — distanţa medie O–O (m). Definiţie operaţională: poziţia primului maximum al funcţiei de distribuţie radială gOO(r) obținută din difracţie neutronică sau rază X.
  • νOH(T) — centroidul benzii de stretching O–H (cm⁻1), extras din fit-ul spectrelor IR/Raman (centroid = ∫ν·I(ν)dν / ∫I(ν)dν pe banda 3000–3800 cm⁻1).
  • EOH(T) — energie asociată modului stretching O–H, **pe moleculă** (J). Formula corectă:
    EOH(T) = h · c · νm, unde νm este frecvenţa în m⁻1 (pentru ν în cm⁻1, νm = ν × 100), h Planck și c viteza luminii.
  • pKw(T) — valoarea pKw a apei la T (experimentală).
  • [H⁺](T), [OH⁻](T) — concentraţii molare (mol·L⁻1) calculate din pKw (pentru apă neutră la T: pH = pKw/2).
  • ne(T) — densitatea „electronică mobilă” (electrons·m⁻3). Definiţii operaţionale: vezi secţiunea 3.3 (variantă A — proxy din pKw, variantă B — determinare din simulări DFT/MD).

2. Formula centrală (operaţională)

Vom defini mai întâi o mărime brută (cu unităţi) și apoi o versiune normalizată, adimensională, pentru comparare între temperaturi.

2.1. Mărimea brută (raw)

Craw(T) := ne(T) · EOH(T) / r(T)^2

Această formă păstrează logica iniţială (produs în numărător, r² în numitor). Are unităţi: (electrons·m⁻3)·(J)·m⁻² = electrons·J·m⁻5 — nu este convenabil pentru comparaţii, de aceea normalizăm.

2.2. Mărimea normalizată (adimensională)

Definim un indice adimensional prin normalizare la o temperatură de referinţă (Tref = 298.15 K recomandat):

C*(T) := Craw(T) / Craw(Tref)

Prin aceasta obținem C*(T_ref) = 1, și toate valorile sunt relative la valoarea de referință — utile pentru testarea raportului R și pentru reproducibilitate independentă de alegerea unor unități arbitrare.

3. Definiţii operaționale şi proceduri de măsurare

3.1. r(T) — distanţa O–O

Măsurare: difracţie neutronică sau rază X (același lot de probe). Extrageţi gOO(r) și determinaţi poziţia primului maxim prin fit (Gauss / Voigt). Raportaţi valoarea și eroarea r ± δr.

3.2. νOH(T) și EOH(T)

Măsurare: FTIR și Raman pe același lot. Calculaţi centroidul benzii O–H (3000–3800 cm⁻1):
ν_centroid = ( ∫_{3000}^{3800} ν·I(ν) dν ) / ( ∫_{3000}^{3800} I(ν) dν ) (units cm⁻1).
Transformare la metri inverse: ν_m = ν_centroid × 100 (m⁻1).
Energia pe moleculă (J): EOH = h · c · ν_m, cu h = 6.62607015e-34 J·s și c = 299792458 m·s⁻1.

3.3. ne(T) — densitatea electronică mobilă (două variante)

Variantă A — proxy din pKw (rapidă, experimentală):

  1. Măsuraţi pKw(T) experimental sau preluaţi din date standard; pentru apă neutră la T, pH = pKw/2.
  2. Calculaţi [H⁺] (mol·L⁻1) = 10^{-pH}.
  3. Conversie la mol·m⁻3: [H⁺] (mol·m⁻3) = [H⁺] (mol·L⁻1) × 1000.
  4. Număr particule/m³: n_ion = [H⁺]·N_A (N_A = 6.02214076e23 mol⁻1).
  5. Definiţi un factor fenomelogic α(T) (0<α≤1) care reflectă fracţia de „electronică mobilă” asociată cu aceste site-uri: n_e(T) = α(T) · n_ion. α poate fi calibrat cu simulări DFT sau cu măsurători de polarizabilitate/conductivitate.

Variantă B — determinare din simulări (preferabil pentru validare): rulaţi calcule DFT/ab-initio pe configuraţii extrase din MD pentru a estima densitatea electronică locală, polarizabilitatea și un proxy pentru electronii disponibili la polarizare/transport. Convertiţi rezultatul la electrons·m⁻3.

4. Propagarea erorilor

Metode propuse:

  1. Analitic (prima aproximaţie) — folosiţi derivata totală pentru Craw: 
    (δC_raw)^2 ≈ (∂C_raw/∂n_e · δn_e)^2 + (∂C_raw/∂E_OH · δE_OH)^2 + (∂C_raw/∂r · δr)^2
    unde ∂C_raw/∂n_e = E_OH/r^2, ∂C_raw/∂E_OH = n_e/r^2, ∂C_raw/∂r = -2·n_e·E_OH/r^3.
  2. Numeric (recomandat) — Monte Carlo: definiţi distribuţii pentru n_e, E_OH, r (Gaussiene cu μ și σ din erori experimentale), generaţi N >= 10⁴ mostre şi calculaţi distribuţia C*(T). Raportaţi media şi intervalul de încredere (95%).

5. Raportul R și testul π

Definiţie: alegem două temperaturi de interes (ex. T₁ = 298.15 K (25°C) și T₂ = 316.15 K (43°C)).

R := C*(T₂) / C*(T₁) = C_raw(T₂) / C_raw(T₁) (deoarece normalizarea se roteşte la T_ref, dar raportul între două temperaturi poate fi calculat direct din valorile brute).

Ipoteza de test: H₀: R = π (sau |R − π| ≤ δ), cu δ ales (ex.: δ = 0.1–0.3).

Procedură: calculaţi distribuţia pentru R folosind Monte Carlo (propagarea erorilor de la n_e, E_OH, r). Determinaţi IC95% al lui R; dacă π nu este inclus, respingeţi H₀.

6. Predicţii falsificabile

Minimal (cantitativ)

  1. Pe același lot de apă pură, măsurile combinate trebuie să dea: R = C*(316.15K) / C*(298.15K) = 3.14 ± 0.3 (valori indicative — praguri pot fi ajustate după studiul de putere).
  2. C*(T) trebuie să aibă o tendință consistentă (ex. creștere între 25–43°C) așa cum rezultă din valorile măsurate.

Extins

  • În soluţii saline cu I controlat, modificările lui C* trebuie să urmeze tranziţii predictibile (prin modificarea α, r etc.).
  • La interfeţe (SFG/ATR-FTIR) C* va fi diferit sistematic faţă de bulk; această diferenţă poate fi comparată cu schimbarea νOH.

7. Protocol experimental recomandat (pilot)

  • Probe: apă ultrapură Type I (aceeași lot pentru toate măsurătorile).
  • Temperaturi: 25, 30, 35, 37, 39.86, 40, 43, 44.7 °C (inclusiv praguri menţionate în model).
  • Măsurători pe același lot și în același timp:
    • densitate ρ(T) — densitometru / pycnometer;
    • pKw(T) — titrare sau metodă electrochimică de înaltă precizie;
    • FTIR & Raman — pentru centroid și integrală banda O–H;
    • difracţie neutronică / rază X — pentru gOO(r) și r(T);
    • conductivitate / mobilitate ionicã — cross-check pentru modelul n_e.
  • Replicare: minim 3 replici independente pentru fiecare temperatură.
  • Analiză: calculaţi C*(T) şi IC (Monte Carlo); testaţi H₀ pentru R.

8. Protocol simulare (MD + DFT)

  1. MD clasice: TIP4P/2005, TIP4P/ε, SPC/E — rulări NPT la temperaturile alese; extrageţi gOO(r), distribuţii de coordonare și estimări spectroscopice (dacă parametrizarea permite).
  2. DFT/ab-initio pe clustere: pentru configuraţii selectate din MD, calculaţi polarizabilitatea şi densitatea electronică locală; folosiţi pentru a calibra α.
  3. Comparaţi n_e din DFT cu proxy-ul pKw şi produceţi funcţia α(T) pentru versiunea A.

9. Pipeline de analiză a datelor (algoritm)

  1. Ingest: fișiere IR/Raman, difracţie, pKw, densitate.
  2. Preproces: eliminare baseline, normalizare intensități, smoothing minim.
  3. Fit IR: multi-Gaussian/Voigt pe banda O–H → Centroid și integrală.
  4. Fit difracţie: extrageți gOO(r) → determinați r ± δr.
  5. Convertiri: pKw → [H⁺] → n_ion → n_e (Variantă A) sau preluare n_e din DFT (Variantă B).
  6. Calcul Craw și C*(T); Monte Carlo N=10⁴ pentru distribuţii și IC.
  7. Test statistic: calculați R și IC95%; decideți acceptare / respingere H₀.
  8. Raportați: figuri C*(T) ± IC, histograma R, analiza de sensibilitate la α.

E_OH_mean, E_OH_sigma = ... # J (pe molecula)
r_mean, r_sigma = ...       # m

samples = []
for i in range(N):
    n_e_i = normal(n_e_mean, n_e_sigma)
    E_OH_i = normal(E_OH_mean, E_OH_sigma)

    C_raw_i = n_e_i * E_OH_i / (r_i**2)

# obtinem media si IC
C_raw_mean = mean(samples)

12. Analiză statistică și putere

Pentru a stabili un prag δ rezonabil pentru compararea cu π, recomandăm simulări de putere folosind estimări ale erorilor pentru fiecare măsurătoare. De exemplu, dacă eroarea relativă combinată a lui C* este ~10%, atunci un δ = 0.3 poate fi de ajuns pentru a avea o putere moderată cu 3–5 replici; calcule exacte se fac prin bootstrap cu distribuţii simulate.

13. Criterii de succes / respingere

  • Succes puternic: R = π ± 0.3 cu IC95% stabil și reproducibil între replici și laboratoare.
  • Succes parţial: trenduri consistente în C*(T) și R în proximitate numerică a π dar cu IC larg.
  • Respingere: π este exclus din IC99% al lui R sau rezultatele sunt inconsistente între replici controlate.

Anexa B — Paşi pas-cu-pas pentru calcul numeric (exemplu complet)

  1. Din spectrul IR: calculaţi ν_centroid (cm⁻1) → ν_m = ν_centroid × 100 (m⁻1).
  2. Calculaţi EOH = h·c·ν_m (J per moleculă).
  3. Din pKw: pH = pKw/2 → [H⁺] (mol·L⁻1) = 10^{-pH} → mol·m⁻3 = ×1000 → n_ion = mol·m⁻3 × N_A.
  4. Calibraţi α (sau folosiţi valoare ipotetică) → n_e = α·n_ion (electrons·m⁻3).
  5. Din difracţie: extrageţi r (m) și δr.
  6. Calcul C_raw = n_e·E_OH / r^2.
  7. Repetati pentru T_ref și normalizaţi: C*(T) = C_raw(T)/C_raw(T_ref).
  8. Monte Carlo pentru erori: folosiţi distribuţii normale pentru n_e, E_OH, r (sau empirice din date) şi calculaţi IC la 95% pentru C* și pentru R.

Anexa C — Pseudocod pentru deconvoluția benzii O–H (sugestie)

# Steps:

4. calculate centroid = sum(component_area * component_center) / total_area
5. output centroid and integrated area
